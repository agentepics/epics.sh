# Examples

This directory will hold:

- local example content and fixtures used by documentation and tests
- sample host adapter setups
- small reference packages used for development, validation, and smoke tests

This directory is not intended to be the public catalog of Epics authored by the
project.

Public sample and curated Epics should live in the separate
`agentepics/epics` repository, while this directory stays focused on local
fixtures and development examples for the monorepo.
